# **查询全体用户**
***

+ **说明**
>获取所有已注册玩家的QQ号

+ **请求url**
>http://127.0.0.1:8088/api/user/user_list

+ **请求方式**
>Get

***
+ **请求参数**

|   请求参数   | 必选  | 参数类型 |   说明    |
|:--------:| :---: | :------: |:-------:|
|  apikey  |  是   |  sring   | Api Key |

+ **请求示例**
~~~ url
http://127.0.0.1:8088/api/user/user_list?apikey=123456
~~~

+ **返回参数**

|   返回参数   | 参数类型  |  说明  |
|:--------:|:-----:|:----:|
|  status  |  int  | 状态码  |
|   data   | array |  数据  |
| error  | sring | 错误信息 |
+ **返回示例**
~~~ js
// 正确示例
{
    "status":200,
    "data":["123456","123456","434658198"]
}
// 错误示例
{
    "status":403,
    "error":"权限不足"
}
~~~
